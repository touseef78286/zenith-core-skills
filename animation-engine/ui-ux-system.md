# Skill: Minimalist UI/UX Aesthetic System

## Purpose
To build sophisticated, high-end interfaces that complement 3D/WebGL experiences.

## Directives
- **Aesthetic:** Prioritize negative space, elegant typography (Inter, Geist, or Serif fonts), and subtle transitions.
- **Library Choice:** Default to Shadcn/UI for components. Use Tailwind CSS for custom styling.
- **Design Principles:** 1. No clutter—only show what is necessary. 
  2. Dark mode by default, or extremely clean light mode with high contrast.
  3. Animations must be smooth (use Framer Motion for UI, not just 3D).
- **Responsiveness:** Ensure perfect mobile-first layout.
- **Constraint:** Never use generic or "default" looking Bootstrap/MUI styles. Everything must look "custom-made" and expensive.
