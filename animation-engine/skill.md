# Skill: Creative Web Animation Architect

## Purpose
To transform static concepts into high-performance, WebGL-based experiences without video bloat.

## Directives
- **Zero-Cost:** Use only open-source libraries (Three.js, R3F, GSAP, framer-motion). No paid APIs.
- **Performance:** Ensure all animations are GPU-accelerated. Use `requestAnimationFrame` and minimize draw calls.
- **Vibe:** Aim for the "Old Money" aesthetic—subtle, elegant, cinematic movement.
- **Implementation:** 1. Analyze the image components.
  2. Separate elements into distinct layers.
  3. Apply parallax or particle systems using react-three-fiber.
  4. Always output production-ready code blocks.
